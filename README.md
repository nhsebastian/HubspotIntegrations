# HubspotIntegrations


### Dependencies
- FastAPI
- uvicorn
- pprint
- hubspot-api-client
- dotenv
- simplejson
- Files
- httpx
- jinja2


### How to run
```
uvicorn main:app --reload
```

https://developers.hubspot.com/docs/reference/api/crm/objects/leads

## To activate venv
- cd HubspotIntegrations
- source hs_venv/bin/activate

## To run webapp
- cd HubspotIntegrations
- activate venv
- uvicorn main:app --reload

## To deactivate venv
- deactivate
